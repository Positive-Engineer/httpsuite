Helpers¶

Collection of helper classes used through the project.

This module contain helper classes that are used throughout httpsuite.

Item¶

class Item(item)[source]¶

Item provides an interface for string, byte, int, and None.

Item is an interface between the original object (str, bytes, int, None) that allows the user to do object-agnostic equality checks (i.e. Item(200) == Item("200") == Item(b"200")), as well as retrieve the original item as either a string or bytes object.

Note

Item will automatically convert the passed item into a bytes object internally.

Parameters

item (Union[str, bytes, int, Item]) – input to be stored.

Headers¶

class Headers(value={})[source]¶

Representation of a HTTP request or response headers object.

Headers, similiar to Item, provides an interface for a HTTP request and response header, allowing easy manipulation, parsing, adding, and returning str and bytes version of itself. bytes version is escaped with \r\n.

Parameters

value (Union[dict, Headers]) – Headers value in dictionary or Headers object.

property string¶

String representation of the Headers.

Returns

String representation of the Headers.

Return type

str

property raw¶

Bytes representation of the Headers.

Note

This method will return Headers with \r\n escape characters.

Returns

Bytes representation of the Headers.

Return type

bytes

TwoWayFrozenDict¶

class TwoWayFrozenDict(data)[source]¶

A frozen dictionary with two-way capabilities.

Interface that locks a dictionary in place after initilization, and provides accessability via key and value.

Note

All the keys and values inside TwoWayFrozenDict are Item objects, which allows easy comparissions to check if an item is inside the TwoWayFrozenDict mapping.

Parameters

data (dict) – Dictionary that will use this interface.
